**EXTENSION**

# `Comparable`

## Methods
### `map(in:)`

Check if the value is in a certain range and, if not, change it to be in the range.
- Parameter range: The closed range.

Change a comparable type so it is in a certain range:
```swift
var value = 10
value.map(in: 0...5)
```

### `map(in:)`

Check if the value is in a certain range and, if not, change it to be in the range.
- Parameter range: The range.
