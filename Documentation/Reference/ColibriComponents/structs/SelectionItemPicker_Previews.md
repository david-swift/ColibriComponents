# `SelectionItemPicker_Previews`

The previews for the ``SelectionItemPicker``.

## Properties
### `previewWidth`

The width of the preview.

### `previewHeight`

The height of the preview.

### `previews`

The previews.

### `selection`

The identifier of the selected item.

### `hover`

The identifier of the hovered item.

### `items`

The presented items.

### `body`

The view's body.
